Десктопное приложение для моделирования очереди в банке методом дискретно-событийного моделирования (SimPy).

Возможности
Настройка количества операторов, интенсивности потока и времени симуляции
Графики в реальном времени: время ожидания, размер очереди, загрузка операторов
Управление скоростью симуляции (1x / 2x / 5x) и пауза
Итоговая статистика по окончании
Запуск
Скачай Bank_Simulation.exe и запусти — установка не требуется.
