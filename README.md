# 🏦 Bank Queue Simulator

**Десктопное приложение для моделирования очереди в банке**  
Метод: **Дискретно-событийное моделирование (SimPy)**  
🎯 Никакой установки — запустил и работаешь.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| **Настройка параметров** | Количество операторов, интенсивность потока клиентов, время симуляции |
| **Графика в реальном времени** | 3 динамических графика: время ожидания, размер очереди, загрузка операторов |
| **Управление симуляцией** | Скорость 1x / 2x / 5x и кнопка **Пауза** |
| **Итоговая статистика** | Автоматический расчёт средней длины очереди, максимального времени ожидания, загрузки операторов и других метрик |

---

## 🚀 Запуск (установка не требуется)

1. Скачайте файл `Bank_Simulation.exe`
2. Запустите двойным кликом
3. Настройте параметры и нажмите **Start**

> ⚠️ **Важно:** При первом запуске Windows SmartScreen может попросить подтверждение — нажмите «Выполнить в любом случае». Приложение не содержит вредоносного кода.
