## 🔬 Новое в версии 1.1.0: Модуль автоматической оптимизации

В данной версии реализован **аналитический блок**, позволяющий проводить многоканальные вычислительные эксперименты:

* **Динамический подбор параметров:** Система автоматически определяет математический минимум операторов и находит "точку баланса", где загрузка персонала находится в целевом диапазоне **70–85%**.
* **Сравнительный анализ:** Программа выполняет серию фоновых симуляций и формирует итоговую таблицу результатов для оценки устойчивости системы.
* **Экспертное заключение:** На основе полученных метрик приложение самостоятельно формулирует рекомендации по количеству рабочих мест для минимизации очередей и исключения простоя персонала.

## ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| **Визуализация** | Динамические графики ожидания и очереди в реальном времени |
| **Авто-оптимизация** | Расчет оптимальной конфигурации СМО по критерию эффективности загрузки |
| **Аналитика** | Расчет коэффициента загрузки мощностей ($U$), средней длины очереди ($L_q$) и времени ожидания ($W_q$) |
| **Управление** | Гибкая настройка стохастических параметров входящего потока |

## 🚀 Запуск и сборка

### Для пользователя:
1. Запустите `Bank_Simulation.exe`.
2. В меню настроек выберите **"Визуализация"** для наблюдения за процессом или **"Провести анализ"** для автоматического поиска оптимального количества операторов.

> ⚠️ **Важно:** При первом запуске Windows SmartScreen может попросить подтверждение — нажмите «Выполнить в любом случае». Приложение не содержит вредоносного кода.

### Для разработчика (Сборка .exe):
Для компиляции проекта используется `PyInstaller`:

```bash
# 1. Установка зависимостей
pip install pyinstaller pyqt5 pyqtgraph simpy numpy

# 2. Сборка исполняемого файла
pyinstaller --noconsole --onefile --name "Bank_Simulation" main.py
